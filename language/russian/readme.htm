Данный архив содержит русский перевод для модуля XOOPS XHLD версии 2.83. Кроме перевода модуля на русский язык я так же выполнил его адаптацию для корректного отображения новостных заголовков в различных русских кодировках: КОИ-8-Р, КОИ-8-У, КОИ-8-РУ, и Windows-1251 (реально я тестировал только отображение новостей в Windows-1251, в других кодировках новостей мне обнаружить просто не удалось). Заголовки со всех сайтов отображались нормально (пример можно посмотреть тут).

Для корректной работы распознавания кириллицы XHLD требуется установленное расширение PHP iconv или поддержка функций iconv на уровне операционной системы. На моем хостинге iconv стоит, и я думаю, на большинстве других - тоже.

Установки никакой не требуется, просто нужно скопировать папку russian из архива в папку языков установленного модуля
